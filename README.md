## âœ¨ Main Features
* **Three User Roles:** Different dashboards for Admin, Contest Creator, and Normal Users.
* **Easy Search:** Find contests quickly by typing the category in the search bar.
* **Secure Payment:** Users can pay the entry fee safely using Stripe.
* **Live Countdown:** Every contest shows a timer to see how much time is left.
* **Winner Board:** A leaderboard that shows users with the most wins.
* **Task Submission:** After paying, users can submit their work links through a modal.
* **Admin Control:** Admins can approve or delete contests and change user roles.
* **Creator Tools:** Creators can add contests, edit them, and pick a winner.
* **Dark & Light Mode:** Switch between dark and light themes easily.
* **User Stats:** See your win history with a simple visual chart in your profile.
